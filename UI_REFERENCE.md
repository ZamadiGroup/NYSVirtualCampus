## Color Palette

### Status Colors

- **Completed**: `bg-emerald-50`, `border-emerald-200`, `text-emerald-700`
- **In Progress**: `bg-blue-50`, `border-blue-200`, `text-blue-700`
- **Locked**: `bg-slate-50`, `border-slate-200`, `text-slate-600`

### Lesson Type Colors

- **Video**: Purple (`text-purple-600`, `bg-purple-100`)
- **Quiz**: Orange (`text-orange-600`, `bg-orange-100`)
- **Reading**: Blue (`text-blue-600`, `bg-blue-100`)
- **Assignment**: Green (`text-green-600`, `bg-green-100`)

### Gradients

- Progress Bar: `from-blue-500 to-emerald-500`
- Background: `from-slate-50 via-blue-50 to-slate-50`
- Footer Tip: `from-blue-50 to-emerald-50`
- Instructor Avatar: `from-blue-400 to-blue-600`

## Interactive States

### Module Header

- **Default**: White background, border
- **Hover**: Slight background change (`hover:bg-slate-50`)
- **Expanded**: Chevron rotates 180Â°
- **Locked**: Cursor not-allowed, reduced opacity

### Lesson Item

- **Uncompleted**: White bg, border, hover effect
- **Completed**: Green background, line-through text
- **Hover**: Background transition

### Buttons

- **Start**: Blue background, white text
- **Review**: Green background, green text
- **Hover**: Darker shade

## Responsive Behavior

- **Mobile**: Single column, full-width cards
- **Tablet**: Same layout, adjusted padding
- **Desktop**: Max-width container (5xl: 64rem)
- **Sticky Header**: Remains visible on scroll

## Animation Details

- Chevron rotation: 300ms ease
- Module expand/collapse: Smooth height transition
- Progress bar fill: 500ms ease-out
- Hover effects: 200ms transition
- Checkmark toggle: Instant

## Accessibility Features

- Clear status indicators
- Descriptive labels
- Keyboard navigable
- Screen reader friendly text
- Sufficient color contrast
- Icon + text combinations
